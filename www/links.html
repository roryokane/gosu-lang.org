Gosu-Related Projects


Ronin

Ronin is an MVC style web framework written in gosu. Unlike other popular web frameworks, there's no reflective magic or arcane XML files. The URL in your browser tells you which method on which class was called to generate the current page. That method contains a call to render the template for the page, and an explicit set of parameters providing the data for the template. With an IDE, you can navigate between all these different pieces with a single click. And best of all, should you make a mistake - misspell a file name, or leave out a parameter - there will be a compilation error, so you'll know right away.

Sounds pretty fantastic, right? It is. Check it out at ronin-web.org

Ronin Plugins

Ronin also has a small but growing number of plugins:

ronin-coffee - support for CoffeeScript in ronin

ronin-less - support for LessCSS in ronin

ronin-ws - support for XML Web Services in ronin

goson - support for JSchema in ronin (Goson is more than a plugin, see below, but can be used to expose JSchema-RPC endpoints)

Aardvark A Sane Build and System Scripting Tool

Aardvark is a gosu-based scripting environment that exposes familiar Ant functionality in a more comfortable programming environment. It also includes integration with Maven for easy dependency resolution and management.

Check it out at vark.github.com

Goson JSON/JSchema support for Gosu

The trend these days to use JSON to express your data is constantly growing and with good reason. We love JSON, but we also love type-safe programming. When we use the goson type-loader, we get both. Everybody wins!

Here is a short list of some of the features of goson:

  • Tools for parsing, validating and producing JSON
  • A Gosu type loader For JSchema (.jsc), JSON (.json) and JSchema-RPC (.jsc-rpc) documents
  • A servlet filter that can be used to publish JSchema-RPC end points
  • A simple HTTP server that can be used to publish JSchema-RPC end points

Do yourself a favor and check out the goson project page for more details and examples.

RoMail JavaMail... Yeah.

JavaMail provides a bunch of great functionality wrapped up in a very... difficult... API

RoMail wraps JavaMail up in a nice, clean API that leverages many Gosu language features. While it is designed to be used with Ronin, it is a stand-alone project and does not depend on Ronin.
