What is Gosu?
  • Gosu is a programming language for the Java Virtual Machine (JVM).
  • Gosu
    • is object-oriented
    • is statically typed
    • is imperative
    • is 100% compatible with Java
    • features type inference
    • supports closures
    • provides simplified generics
    • is being used in production by multi-million dollar companies around the globe
