Gosu 0.9

The latest and greatest

Download Gosu 0.9

Released January 25th 2012, the 0.9 release is the latest stable release.

  • gosu-thin-0.9-4.zip (1.7 MB)
  • gosu-thin-0.9-4.tgz (1.7 MB)
  • gosu-full-0.9-4.zip (13 MB)
  • gosu-full-0.9-4.tgz (13 MB)

Installation is dead simple. Just unzip/extract your download and add the bin directory to your PATH. Be sure you have at least Java 1.6 installed as well as valid JAVA_HOME environment variable setup, too.
